**sync.py** — single flat script with these components in order:

1. **Imports:** `re`, `json`, `os`, `unicodedata` from stdlib; `Path` from pathlib; `datetime` from datetime; `dataclass` from dataclasses; `frontmatter`; `typer`

2. **Constants:**
   - `VERSION_SUFFIX = re.compile(r'-\d+$')`
   - `APPLE_DATE_FMT = "%A, %d %B %Y at %H:%M:%S"`

3. **Dataclasses:**
   - `NoteFile` with fields: `account: str`, `folder: str`, `title: str`, `md_path: Path`, `attachment_dir: Path | None`
   - `ParsedNote` with fields: `title: str`, `body: str`, `created: datetime | None`, `modified: datetime | None`, `attachment_dir: Path | None`

4. **`is_canonical(path: Path) -> bool`** (per PARSE-01):
   - Returns `not VERSION_SUFFIX.search(path.stem)`

5. **`discover_notes(export_dir: Path) -> tuple[list[NoteFile], int]`** (per PARSE-01, PARSE-05):
   - Walk `export_dir` two levels deep: account dirs then folder dirs
   - For each `.md` file in folder dirs, filter with `is_canonical()`
   - For canonical files, check for sibling attachment directory: `folder_dir / md_file.stem` — set `attachment_dir` if `.is_dir()` is True, else None
   - Count and return versioned files separately
   - Use `sorted()` on all `.iterdir()` and `.glob()` calls for deterministic order

6. **`parse_apple_date(value: str) -> datetime`** (per PARSE-03):
   - `return datetime.strptime(value.strip(), APPLE_DATE_FMT)`

7. **`parse_note(note_file: NoteFile) -> ParsedNote`** (per PARSE-04):
   - Use `frontmatter.load(str(note_file.md_path))` to get post object
   - Extract title: `post.get("title", note_file.md_path.stem)`, then `unicodedata.normalize("NFC", title)`
   - Extract created/modified: call `parse_apple_date()` on string value, wrapped in try/except returning None on failure
   - Return `ParsedNote` with `body=post.content`, the parsed title, dates, and `note_file.attachment_dir`

8. **`clean_body(body: str, title: str) -> str`** (per PARSE-02):
   - If `not body`: return `body`
   - Split into lines; if `lines[0].strip() == title`: return `"\n".join(lines[1:]).lstrip("\n")`
   - Otherwise return body unchanged

9. **`load_state(state_path: Path) -> dict`** (per STATE-01, STATE-02):
   - If file doesn't exist: return `{"version": 1, "items": {}}`
   - Try `json.loads(state_path.read_text(encoding="utf-8"))`
   - On `json.JSONDecodeError` or `OSError`: print warning, return fresh state

10. **`save_state(state: dict, state_path: Path) -> None`** (per STATE-03):
    - Write to `state_path.with_suffix(".json.tmp")` using `write_text(json.dumps(state, indent=2, ensure_ascii=False), encoding="utf-8")`
    - Then `os.replace(str(tmp), str(state_path))`

11. **`run_parse_only(export_dir: Path) -> None`** (per D-05, D-06):
    - `state_path = export_dir / "nuclino-state.json"` (per D-01)
    - Load state, discover notes
    - For each canonical note: check if `rel_path` (relative to export_dir) is already in `state["items"]` — if so, tally and skip (per STATE-02)
    - Parse note, clean body
    - If `not body.strip()`: print `f"skipped empty note: {parsed.title}"` (per D-03), record in state with `"status": "skipped_empty"` (per D-04)
    - Otherwise: record in state with `"status": "parsed"`, `"title"`, `"created"` (isoformat or None), `"modified"` (isoformat or None)
    - Call `save_state()` after EACH note (not at end of run) — per anti-pattern warning in research
    - Track accounts (set), folders (set of `account/folder`), skipped_empty count, parsed_count
    - Print summary per D-06: `f"Found {parsed_count} canonical notes across {len(accounts)} accounts, {len(folders)} folders. {skipped_empty} skipped (empty). {versioned_count} versioned snapshots ignored."`

12. **Typer CLI:**
    - `app = typer.Typer()`
    - `@app.command()` with parameters:
      - `export_dir: Path = typer.Option(Path.home() / "Desktop" / "NotesExport", help="Path to Apple Notes export directory", envvar="NOTES_EXPORT_DIR")`
      - `parse_only: bool = typer.Option(False, "--parse-only", help="Parse export and print summary without importing")`
    - If `parse_only`: call `run_parse_only(export_dir)`
    - Else: `typer.echo("Import not yet implemented (Phase 2+)")` and `raise typer.Exit(1)`
    - `if __name__ == "__main__": app()`
